# BaksDev Nginx Unit

![Version](https://img.shields.io/badge/version-7.0.7-blue) ![php 8.2+](https://img.shields.io/badge/php-min%208.1-red.svg)

Модуль управления сервером Nginx Unit

## Установка

``` bash
$ composer require baks-dev/nginx-unit
```

## Дополнительно

Тесты

``` bash
$ php bin/phpunit --group=nginx-unit
```
